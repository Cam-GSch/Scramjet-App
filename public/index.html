About Eclipse

Eclipse is a sleek frontend for the Scramjet web proxy framework by Mercury Workshop. It uses service worker interception to route your web traffic through a secure backend, enabling access to restricted content while maintaining privacy.

Scramjet is the successor to Ultraviolet — an advanced interception-based proxy with CAPTCHA support, WASM-accelerated HTML rewriting, and full compatibility with sites like Google, YouTube, Discord, and more. This frontend requires a configured Scramjet backend to function.
